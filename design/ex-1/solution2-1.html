Font Weights Applied

  • Body Text: Roboto (Sans-Serif), normal weight.
  • Main Heading (h1): Merriweather (Serif), bold weight.
  • Subheadings (h2, h3): Merriweather (Serif), normal weight.
  • Paragraph and List Text: Roboto (Sans-Serif), light weight.
  • Code Blocks: Source Code Pro (Monospace), normal for preformatted text; Semi-bold for inline code.
