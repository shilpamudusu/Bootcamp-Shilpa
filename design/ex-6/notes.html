Key Findings on Current Indian Railways Website (IRCTC)

  1. User Interface (UI) Issues

    • Cluttered Design : Overloaded homepage with too many options and information
    • Outdated Aesthetics:Lack of modern design elements (e.g., flat design, minimalist approach).
    • Inconsistent Layouts:Pages often vary in design, creating a jarring user experience.

    2. User Experience (UX) Problems

    • Complex Navigation:Too many nested menus and options, making it difficult for users to find what they need.
    • Slow Loading Times:High traffic and heavy elements cause delays in page response.
    • Non-Intuitive Workflow:Booking tickets or checking train status involves too many steps.

    3. Accessibility Concerns

    • No Proper Support for Screen Readers:Accessibility features are limited.
    • Small Fonts and Poor Color Contrast:Affects readability, especially for visually impaired users.
    • Language Limitations:No prominent multilingual support despite a diverse user base.
