React useState Hook Notes

  • Import `useState`: Ensure you import `useState` from React when using it in functional components.
  • React Hook: `useState` allows you to manage state in a functional component.
  • Error Resolution: If TypeScript shows "Cannot find name 'useState'", check if `useState` is imported correctly.
  • React Installation: Ensure that React and its types (`@types/react`, `@types/react-dom`) are installed to avoid errors.
