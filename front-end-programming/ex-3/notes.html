Exercise 3: Starting with Vite - Notes

Objective

The goal of this exercise was to learn how to scaffold, configure, and build modern web projects using Vite, along with enhancing a student management dashboard by loading data from various formats (JSON, CSV), parsing it, and displaying it in a table.

Challenges Encountered

  • Setting Up Vite for JavaScript and TypeScript: Initial confusion about configuring two separate Vite projects for JavaScript and TypeScript.
  • Data Fetching and Display: Handling asynchronous data fetching and rendering from JSON and CSV formats.
  • Handling Data Formats: Managing different data formats (JSON and CSV) and ensuring smooth parsing and display.
  • CSS/SCSS Styling: Understanding SCSS loops for generating margin utility classes and ensuring Vite processes SCSS correctly.
  • Modularization: Breaking the code into separate modules for data loading, parsing, and rendering to keep the project organized.

How the Outcome Was Achieved

  • Vite Setup: The Vite project was successfully initialized for both JavaScript and TypeScript, with the development server running smoothly.
  • Data Loading and Display: Data was fetched asynchronously from JSON and CSV files, parsed, and displayed in a table using React.
  • Modularization of Code: Code was modularized, separating data fetching logic from rendering logic, improving readability and maintainability.
  • SCSS Utility Classes: Margin utility classes were implemented in both CSS and SCSS, with the SCSS approach using loops for efficiency.
  • Styling with SCSS: SCSS files were processed successfully, applying dynamic margin classes generated by loops.

Final Outcome

The student management dashboard was successfully built, with data displayed from both JSON and CSV formats. The project was organized in a modular way, allowing for easy maintenance and further development. Additionally, the use of Vite's built-in support for SCSS helped streamline the styling process, creating dynamic margin utility classes with minimal effort.

Note: This exercise provided hands-on experience with Vite, modern JavaScript and TypeScript tools, data fetching, and modular project design. It also demonstrated how to manage CSS and SCSS effectively within a Vite project.
