Exercise 3

Objective: Implement structured logging using winston library.
Level 1:
  • Create logger with log levels (info, warn, error).
  • Log events like "Program started" and "Operation successful".
  • Log fatal error for division by zero.
Exit Criteria: Logs are output to the console with appropriate log levels.
Level 2:
  • Add file transport to log messages.
  • Customize log format with timestamps.
Exit Criteria: Logs are written to a file with timestamps. Format should be clear and readable.
