Project Notes

Level 1: Project Setup

  • Initialize Node.js Project: Use npm init -y to quickly set up a project.
  • Scripts:
    • "start": Runs the main program (e.g., node index.js).
    • "dev": Adds debugging output (DEBUG=*) for development.
  • Test Commands:
    • Run npm start for normal execution.
    • Run npm run dev for debug mode.

Level 2: Styling Output and Auto-Restarts

  • Chalk Package:
    • Install: npm install chalk.
    • Usage: Styles console output. 
      const chalk = require('chalk');
console.log(chalk.green("Success!"));
  • Nodemon for Development:
    • Install: npm install --save-dev nodemon.
    • Dev Script: Auto-restart program on changes using "dev": "nodemon index.js".

Level 3: Prestart Script

  • Prestart Script: Runs npm install before the start script.
    • Add to package.json: "prestart": "npm install".
  • Testing:
    • Delete node_modules: rm -rf node_modules.
    • Run npm start to verify automatic dependency installation.
